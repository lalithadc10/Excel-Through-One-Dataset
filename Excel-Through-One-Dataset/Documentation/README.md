#  Documentation

Welcome to the documentation section of the **Excel Through One Movie Dataset** project.

This folder contains supporting information about the repository, project organization, and learning journey followed during the 15-Day Excel Challenge.

---

##  Repository Structure

The repository is organized into multiple sections to make navigation simple and beginner-friendly.

- 📁 Assets
- 📁 Dataset
- 📁 Documentation
- 📁 Final Project
- 📁 Day 01 – Day 15

---

##  Project Objective

The objective of this project is to learn Microsoft Excel through one consistent movie dataset while solving practical business problems.

Each day focuses on a different Excel feature and demonstrates how it can be applied in real-world business scenarios.

---

##  Learning Roadmap

The challenge covers:

- Day 01 – XLOOKUP vs VLOOKUP
- Day 02 – Relative, Absolute & Mixed References
- Day 03 – IF & IFS
- Day 04 – Text Functions
- Day 05 – Date Functions
- Day 06 – Pivot Tables
- Day 07 – Dashboard & Conditional Formatting
- Day 08 – SUMIFS
- Day 09 – INDEX + MATCH
- Day 10 – Power Query
- Day 11 – Dynamic Arrays
- Day 12 – Goal Seek & Scenario Manager
- Day 13 – Dependent Dropdowns
- Day 14 – Debugging VLOOKUP
- Day 15 – When Excel Isn't Enough (Power BI Transition)
