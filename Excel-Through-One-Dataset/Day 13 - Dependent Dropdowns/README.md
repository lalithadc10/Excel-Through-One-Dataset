# 📅 Day 13 – Dependent Dropdowns using INDIRECT

---

## 🎯 Business Scenario

A movie production company allows users to search for movies by first selecting an Industry and then choosing a Studio.

Instead of displaying every studio in a single dropdown, the analyst wants the Studio list to change automatically based on the selected Industry, making the dashboard easier to use and reducing incorrect selections.

---

## 📌 Business Problem

Create dynamic dependent dropdown lists that improve data accuracy, simplify user input, and provide a better reporting experience.

---

## 🛠 Excel Features Used

- Data Validation
- Named Ranges
- INDIRECT Function
- Dependent Dropdown Lists
