# 🎬 Final Project – Movie Analytics Dashboard

---

## 📖 Project Overview

This project brings together all the concepts learned during my **15-Day Excel Challenge** into a single business solution.

Using one movie dataset, I built an interactive Excel dashboard that helps analyze movie performance, financial metrics, studio performance, and business insights.

Instead of learning Excel functions in isolation, this project demonstrates how multiple Excel features can be combined to solve real-world business problems.

---

## 🎯 Business Scenario

A movie production company wants an interactive dashboard to monitor business performance.

Management needs quick answers to questions such as:

- Which movies generated the highest revenue?
- Which studios produced the most profitable movies?
- How many blockbuster movies were released?
- Which genres perform better?
- What is the average movie rating?
- How does budget impact revenue?

Instead of manually analyzing thousands of records, the dashboard provides instant insights.
