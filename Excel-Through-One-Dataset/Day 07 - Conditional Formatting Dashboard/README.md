# 📅 Day 07 – Conditional Formatting & Interactive Dashboard

---

## 🎯 Business Scenario

A movie production company wants a single dashboard to monitor key business metrics such as revenue, profit, movie performance, and category-wise insights.

Instead of analyzing thousands of rows manually, management needs a visual dashboard that highlights important trends and exceptions for faster decision-making.

---

## 📌 Business Problem

Transform raw movie data into an interactive dashboard using Conditional Formatting, Pivot Tables, Charts, and KPI cards to make business insights easy to understand.

---

## 🛠 Excel Features Used

- Conditional Formatting
- Pivot Tables
- Pivot Charts
- KPI Cards
- Slicers
- Dashboard Design
