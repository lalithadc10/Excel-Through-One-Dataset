# 📅 Day 14 – Debugging VLOOKUP Formula

---

## 🎯 Business Scenario

A movie production company maintains separate datasets for movie details and financial information.

A Business Analyst uses VLOOKUP to retrieve movie budgets based on Movie ID. Later, a new column is inserted into the lookup table, causing the formula to return incorrect results without displaying an error.

The analyst needs to identify the issue, debug the formula, and apply a reliable solution.

---

## 📌 Business Problem

Troubleshoot VLOOKUP errors caused by changes in the lookup table structure and understand how to build more reliable lookup solutions.

---

## 🛠 Excel Functions Used

- VLOOKUP
- IFERROR
- XLOOKUP (Comparison)
- INDEX + MATCH (Comparison)
