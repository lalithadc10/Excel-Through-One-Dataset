# 📅 Day 10 – Power Query

---

## 🎯 Business Scenario

A movie production company receives monthly movie data from different departments in separate Excel files.

Instead of manually copying and cleaning data every month, the analyst needs an automated process to combine all files into a single clean dataset for reporting.

---

## 📌 Business Problem

Automate data preparation by combining multiple Excel files, removing duplicate records, and creating a clean dataset ready for analysis.

---

## 🛠 Excel Features Used

- Power Query
- Get Data from Folder
- Combine Files
- Remove Duplicates
- Close & Load
